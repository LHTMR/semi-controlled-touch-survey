# Process

Ran the script using the file word_frequencies_grouped.csv.

I generated two files using two distinct scripts: 
1) prepare_words.py to get the words_filtered (both in txt and csv). This contains both keywords and all words of a group
2) prepare_words_main_only.py to get the words_filtered_main_only file (both in txt and csv). This only contains group keywords and this is the one I used for wordnet.
